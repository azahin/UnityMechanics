# UnityMechanics
Just a place to store mechanics for future use

# Ground Rules for Modularity
Create a new folder for each mechanic, and make a new scene in that folder for testing said mechanic

Cause and Effect design. The causes should not have the effect attached to it. For instance, a button pushed opens a door. The code to open the door shouldn't be on the button, it should be on the door's code.
